# Сложно сосредоточиться

Учебный адаптивный сайт о концентрации внимания. Проект выполнен на HTML, CSS и JavaScript без сборщика и сторонних библиотек.

## Репозиторий
Ссылка на репозиторий:
[https://github.com/vanished2/slozhno-sosredotochitsya-ad](https://github.com/vanished2/slozhno-sosredotochitsya-ad)

## Стек

- HTML5
- CSS3
- JavaScript
- CSS custom properties
- Адаптивная верстка
- Локальные шрифты IBM Plex Mono

## Функциональность

- адаптивная верстка для мобильного, планшетного и десктопного экранов;
- светлая, темная и автоматическая темы;
- сохранение выбранной темы в `localStorage`;
- декоративная галерея изображений;
- семантичная HTML-структура.

## Как запустить

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python -m http.server 8000
```

После запуска откройте в браузере:

```text
http://127.0.0.1:8000/
```
